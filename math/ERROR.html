절단 오차(trancation error)

절단 오차는 정확한 수학적 표현이나 공식 대신 근사식으로 표현할 때 사용된다.

테일러 급수(Taylor series)

  • e^x 미분시, e^x 나옴
  • f(x) = e^x
  • e^x = 1+ x+ x^2/2!+x^3/3!+x^4/4!+ ...+x^n/n!
  • 비판정법 : Sigma(an) = lim((a(n+1))/an) <1 이면 무한 급수는 수려므 1보다 크면 발산한다.

* ln(x+1) 미분시 ===> 1/(x+1)

맥라우린 급수 : 테일러 급수에서 주의해야 될 사항은 좌변과 우변이 모든 x에 대해 같은 것이 아니라 x = a 근처에서만 성립한다는 점이다.

  • T(x) = Sigma(f(a)/!n*(x-a)^n) = f(a)+f'(a)*(x-a)+1/2*f''(a)*(x-a)^2+1/6*f''(a)*(x-a)^3+ ... 일때, f는 a에서 n계의 도함수를 말한다.
  • e^x = 1+ x+ x^2/2!+x^3/3!+x^4/4!+ ...+x^n/n!
